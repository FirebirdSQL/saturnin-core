
#######################
Saturnin_ CORE services
#######################

The core set of services for Saturnin currently contains next services:

* Text file reader, writer and linefilter microservices.
* Binary data file reader and writer microservices.
* Protobuf printer, filter and aggregator microservices.
* Firebird log parser and from server provider microservices.
* Firebird trace parser and session provider microservices.

.. seealso::

   For installation instructions, see the Saturnin_ documentation.

.. note:: Requires Python 3.8+

Saturnin development is sponsored by IBPhoenix_.
